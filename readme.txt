Display recent YouTube channel or playlist videos in a responsive Elementor grid without an API key.

== Description ==

Elementor YouTube Grid uses public YouTube RSS feeds and WordPress transient caching. It provides responsive layout, thumbnail, metadata, hover, lightbox, and typography controls.

The plugin connects to YouTube for RSS data, thumbnails, links, and embedded playback. See the External service and privacy section in README.md before production use.

== Installation ==

1. Activate Elementor.
2. Activate Elementor YouTube Grid (RSS).
3. Edit a page with Elementor.
4. Add YouTube Grid (RSS) and configure a channel or playlist ID.
